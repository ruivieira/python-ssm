# Python state-space models (`pssm`)

A Python package for state-space models. Basic usage in the [example notebooks](docs/state_space_models.ipynb).

# Features

- [ ] Dynamic Generalised Linear Models
  - [x] Normal DLM
  - [x] Poisson DLM
  - [x] Binomial DLM
  - [x] `iterator`
- [ ] Model composition
  - [x] Locally constant
  - [x] Locally linear
  - [x] Cyclic Fourier
  - [x] ARMA(_p_)
- [ ] Multivariate
  - [x] Composite DGLMs
  - [ ] Multivariate Gaussian
- [ ] Filters
  - [x] Kalman filter
